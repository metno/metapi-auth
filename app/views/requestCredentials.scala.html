Request New API Client ID

To gain access to the data.met.no web services, register with your email address to generate a new API Client ID.

@form(action = controllers.routes.AuthorizationController.newCredentialsRequest, 'class -> "new_clientid") { @inputText(tokenRequestForm("email"), 'id -> "user_email") }

We require an API client ID in order to provide all users of this free service with as stable and fast performance as the backend allows. The email given here will be used to contact you to inform about or resolve API maintenance issues.

Submit your email address

  • Enter your email address into the field above.
  • Click on the Submit button.

Save your API client ID

  • You will be taken to a page with an API Client ID and a client secret. Save both of these codes in a secure location.

Start using data.met.no

  • Go to the API documentation and copy your API Client ID into the Client ID form. Experiment with the API resources to mine data from data.met.no.
